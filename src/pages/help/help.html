Setting Pi surveillance system location
  • Go to DASHBOARD and select SET PI LOCATION to open maps page.
  • At the top right corner of the maps page, find and click on the navigation icon.
  • An action sheet displays, click RELOAD to set new Pi location, DELETE to clear saved location, or CANCEL to return back to maps
 How to view captured image from the Pi surveillance system
  • Go to DASHBOARD and select VIEW PI IMAGE to view the last image taken by the Pi surveillance system.
 Alerting the Police or Fire Service
  • Go to DASHBOARD and select ALERT POLICE or ALERT FIRE to send sms from the app
  • Enter police/fire service mobile number in the first text field.
  • In the second text field, type message to be sent to police (append Pi location).
  • Tap the SEND button to send the sms to the police/fire service.
  • Send alert to Police/Fire Service containing user location. Done!
